CSS3 Transforms

Transform!

This property adds a 2D or even 3D transformation to an element.
Features include: rotate, translate, scale, and skew.

Rotate

Moves an element clock or counter clockwise according to your value.
Don't forget to set the word "deg" at the end of your value

rotate(80deg);


rotate(280deg);

Rotate 3D

Using rotateX -> Rotates the element through its X axis

rotateX(80deg);

Using rotateY -> Rotates the element through its Y axis

rotateY(280deg);

Rotate(1080deg); -> This will cause a fast rotation

Scale

It will increase an element's width

Using scaleX(5) increases the width thorugh the X axis

Scale 3D

Use scale to add 3D styling

scale3d(3,2,1) x,,y,z values

Skew

This method skews the element in its X and Y axis according to the values given

skew(30deg,20deg);

Skew 3D

3D effects can be applied through X and Y axis

skewX(10deg); skewY(160deg);
