CSS Animations

We can use animation to change an element's values with timing, movement, and dimension.

Keyframes

This is a set of instructions where you specify the css properties.
You also specify the different moments when the animation will start using percentages
like 25%, 50%, 75%, indicating the different animation moments.

Moving Objects and Colors

We will set an object to move to the right while changing its color

This is the keyframe configuration:

In each keyframe, the left property directs the object to the left side of the DOM

@keyframes move {

0% {background-color: white; left: 0px;}
20% {background-color: rgb(233, 221, 221); left: 100px;}
40% {background-color: rgb(212, 206, 206); left: 200px;}
60% {background-color: rgb(163, 159, 159); left: 300px;}
80% {background-color: rgb(110, 108, 108); left: 400px;}
100% {background-color: rgb(70, 68, 68); left: 500px;}

We now turn a square into a circle while it's rotating
During the different fotograms(0%, 50%, etc), we will apply a border-radius property
that will alter the square's borders into a circle while changing the backgorund color
and while applying a transform rotation of 0 and degrees at fotograms 50, 75, 100

0%{border-radius: 50%;}

50%{border-radius: 0%; background-color: white; transform: rotate(0deg);}
75%{border-radius: 0%; background-color: white; transform: rotate(90deg);}
100%{transform: rotate(90deg);}
