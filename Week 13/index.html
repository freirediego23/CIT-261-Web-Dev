Javascript Events

Events are actions applied to the page. These can be triggered through Javascript
which can also manipulate the page and improve it.

Onload

This Event handler will automatically activate the code once the page is load.
We are going to load a special greeting for our user!

Clicking

Using the button element, we can create a button along with the "onclick" atrribute
This will activate a Javascript function if the JS file is obviously link to the HTML

When the user "clicks" the button, the event will be triggered hence the onclick name

Working when user clicks the square.

Hover & Hovering Out

This event will trigger when the user hovers the square.
This is done thanks to the "onmouseover" property in the HTML

onmouseover will trigger the JS function that will add styling in this case!

onmouseout will trigger an additional JS function with the original dimensions of the square
once the user stops hovering the square

id="output2" onmouseover="hover()" onmouseout="reset()">

Onmouseover JS Function

document.getElementById('output2').style.backgroundColor ="green";
document.getElementById('output2').style.border = "4px solid black";
document.getElementById('output2').style.height = "350px";
document.getElementById('output2').style.width = "350px";
document.getElementById('output2').style.transform = "rotateY(180deg)";
document.getElementById('output2').style.transition = "3s";

Onmouseout JS Function

function reset() {
document.getElementById('output2').style.width = "200px";
document.getElementById('output2').style.height = "200px";
document.getElementById('output2').style.backgroundColor = "yellow";
document.getElementById('output2').style.transform = "rotateY(-180deg)";
document.getElementById('output2').style.border = "2px solid black";
}

Keydown

onkeydown will trigger the related function once any key is pressed!
In this case, we added a small input section to recognize our keyboard activity.
The square should decrease its size using a scale transform and turn into a circle.


Type anything to activate!

Touchstart Event

This event will trigger the JS code when an user touchs a determined element of the page.

This square will turn green once you touch it! (Valid only for mobile devices)

Thanks to the "ontouchstart" event, we can trigger several JS functions in the DOM.

We will trigger the ontouchend property to apply another function when someone removes the finger
from the screen.
