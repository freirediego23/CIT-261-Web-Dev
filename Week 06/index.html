HTML Video Tag

The video tag allows HTML to embed videos:
It is very useful if we want to share media.
We can specify the height and width of the video box along with the type ="" to set our browser.

There are additional attributes to configurate our video tag:

*Autoplay: Sets the video to start after it's ready.
*Controls: Adds controls like volume and play buttons to the video.
*Loop: Sets the video to repeat itself over and over again.
*Muted: Mutes the audio output.

HTML Canvas

Canvas is used to draw using javascript functions for placement and styling.

For this process, it is important to learn some attributes in JS
getElementById links our canvas element into a variable
that is called along with .getContext("2d"); this specifies
that our work will limit to 2d drawing.

Creating a Circle

If the variable storing the .getContext("2d"); is named ctx
We use -> fillStyle to add a color to our draw -> ctx2.fillStyle ="red";
We use -> beginPath to indicate where our draw begins -> ctx2.beginPath();
We use -> .arc to create an arc or curve used for circles -> ctx2.arc(195,130,100,0,Math.PI*2,true);
.arc structure is (x,y,radius,starting angle,ending angle)
We use -> .fill() to set the drawing -> ctx.fill();
