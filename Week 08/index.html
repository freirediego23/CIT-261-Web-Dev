Using CreateElement

CreateElement will create a new element in HTML.
You need to determine the element. This works in combination
with appendChild()

Prepare a var to contain the function -> var child = document.createElement("b");
This will allow us to add text for the element whenever necessary -> child.innerText = "Object created!";
Typing appendChild will append the element you create as the last node to create more elements
document.getElementById('container1').appendChild(child);

The "b" stands for "bold" to create a bold text element

We will combine create element, remove child, and insert before commands.

First create an element using create element. It will be a div element

For this case, we add set atrribute to our element variable -> choco.setAttribute("id", "chocolate");
This will set an id for each div element created to add the dark chocolates!

To remove an element, first specify it's location in the DOM -> var remove = document.getElementById('chocolate');

We use removeChild to remove a child node of an specific element -> return remove.parentNode.removeChild(remove);

This will create the other chocolate

This is a normal create element code, but it was set to be inserted before the first chocolate

function insert() {
var choco1 = document.getElementById('chocolate').lastChild;

var choco2 = document.createElement("div");
choco2.setAttribute("id", "chocolate2");
document.getElementById('container2').appendChild(choco2);

choco2.insertBefore(choco1, choco2);
}

The insertBefore will insert a node as a child before an existing element -> choco2.insertBefore(choco1, choco2);
