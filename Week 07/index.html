JSON Parse

JSON parse turns a string into a Javascript oject
Most requests from a web server come as a string which can be used with JSON.parse()

We have a recipe string:

'{"food":"Ceviche", "recipe":"fish", "tomato", "cucumber"}'

Using Json.parse(info);
We can now access each object's property -> document.getelementbyid("output") = info.food + " " + info.recipe;
