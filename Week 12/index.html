Local Storage Api

Storing and Retrieving Simple Data

Local storage is possible by using the "localStorage" property
It will save our data locally without expiring. It can also include arrays or simple words.

This is my stored ID card

We define the storage variable name and the content:

localStorage.setItem("info", "Name: Diego & ID: 235412");

Set item -> defines (name, content)

To retrieve the stored item, we use getItem("name of variable")
document.getElementById('ID').innerHTML = localStorage.getItem("info");


Using Arrays

To store arrays, these must be converted to a string using "stringify"

Having: var fruit = ["apple", "banano", "grape", "watermelon", "cherry", "strawberry"];
localStorage.setItem("list", JSON.stringify(fruit));

The next step is to retrieve the stored string -> var retrievelist = localStorage.getItem("list");

The array can be turned into an object by using JSON.parse
var fruits2 = JSON.parse(retrievelist);
We can now use any part of the string we got!

Using Objects

Using an object -> var instrument = new Object();
instrument.name = "Guitar";
instrument.purpose = "Play some nice music";

Storing the object -> localStorage.setItem('firstname', instrument.name);
Retrieving the object -> var instrumentName = localStorage.getItem('firstname');
Displaying the object -> document.getElementById('output1').innerHTML = instrumentName;
