CSS3 Transitions

Using Transition

It will allow us to gradually change an element with the values we set.

Here are some examples:

Give Color

Click the button to activate the transition effect. You will see how the square
changes to orange with yellow borders gradually

Changing Dimensions

We can alter the dimensions of a div elements and then adding a transition.
For this example, width and height changes will be triggered once you hover the square.

This is the div triggered with hover condition -> #block2:hover {
Height values -> height: 250px;
Width values -> width: 400px;
}

Hide and seek

Setting the visibility property to "hidden" once you hover the square.
This element will disappear but it is still present in the DOM
